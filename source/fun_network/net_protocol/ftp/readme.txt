/*
 ftp(file transfer protocol )文件传输协议 基于TCP   21端口-命令端口
*/

1.工作模式
PORT  主动方式      由客户端提供数据传输端口

1.客户端向服务器发送连接请求(21-port)。
2.当要传输数据时，客户端发送PORT命令，告诉服务器要传输数据。
3.服务器从20端口向客户端发起连接请求，建立链路来传输数据

PASV  被动方式      由服务器端提供数据传输的端口

1.客户端向服务器发送连接请求(21-port)
2.当要传输数据时，服务器发起PASV命令(指定一个端口发给客户端)，告诉客户端要连接某个端口
3.客户端向服务器的xxx端口发起连接。建立链路来传输数据
